**Cloud Parkour**

Criei este repositório como parte do processo seletivo da KodLand!
Trata-se de um jogo de plataforma desenvolvido em Python, utilizando o Pygame Zero e a classe Rect para manipulação de colisões e áreas de jogo.

--Tecnologias Utilizadas--

Python 

Pygame Zero

Pygame Rect

--Funcionalidades--

Sistema de colisão entre o jogador e plataformas

Múltiplos estados de jogo (menu, jogo, pausa, etc.)

Controle de música e sons

Estrutura modular e comentada para fácil manutenção

--Dica--

Para melhor visualização dos comentários e anotações no código, recomendo o uso da extensão “Better Comments” desenvolvida por Aaron Bond (disponível no VS Code).
